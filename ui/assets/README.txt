# 应用图标说明

此目录用于存放应用图标文件。

为使应用显示正确的图标，请在此目录中放置名为`icon.ico`的图标文件。

## 图标需求

- Windows系统需要`.ico`格式
- macOS系统需要`.icns`格式
- Linux系统可以使用`.png`格式

## 获取图标的方式

1. **使用在线转换工具**：
   - [Convertio](https://convertio.co/) - 可将图片转换为适合的图标格式
   - [ICO Convert](https://icoconvert.com/) - 专门用于创建ICO文件
   - [CloudConvert](https://cloudconvert.com/) - 支持多种格式转换

2. **使用推荐的免费图标资源**：
   - [Icons8](https://icons8.com/icons/set/book) - 提供免费的书籍和小说图标
   - [IconArchive](https://iconarchive.com/search?q=book) - 多种书籍相关图标
   - [Flaticon](https://www.flaticon.com/search?word=book) - 丰富的扁平化图标

## 创建图标的推荐尺寸

请确保图标包含以下尺寸以获得最佳显示效果：
- 16x16
- 32x32
- 48x48
- 64x64
- 128x128
- 256x256

## 图标替换方法

1. 将您的自定义图标文件命名为`icon.ico`
2. 将文件放置在此目录中（`novel_generator/ui/assets/`）
3. 重启应用程序以应用新图标
